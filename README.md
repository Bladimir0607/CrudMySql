# CRUD API con C# y MySQL

Proyecto académico que implementa un CRUD básico utilizando ASP.NET Core, Entity Framework Core y MySQL.

## Tecnologías
- C#
- ASP.NET Core
- Entity Framework Core
- MySQL
- Swagger
- Visual Studio 2022

## Endpoints
- POST /api/Tareas
- GET /api/Tareas
- PUT /api/Tareas/{id}
- DELETE /api/Tareas/{id}
